### task

#### Tasks.todo()  

add task  

_Example_

````js

Tasks.todo('task#1')

````

#### Tasks.done()  

declare task it's done  

_Example_

````js

Tasks.todo('task#1')

````

#### Tasks.Tasks(options, options.done)  
- _options_ \<Object\>  
- _options.done_ \<function\> callback when all tasks are completed 
simple parallel tasks manager
